클래스안에 클래스 선언 가능, 
-----------------------------------------------------------

* 자바 버전 : lts버전이 롱텀. 그 나머지는 테스트 버전.
* graalVM은 롱텀서비스버전 설치해줌. 
* java 11 api검색해서 나오는 페이지 즐겨찾기 추가.
* 클래스를 분류한게 패키지, 패키지를 분류한게 모듈. 
* 이클립스에서 Math입력하고 컨트롤 클릭하면 메스 클래스 볼 수 있음. 
메스와 관련된 메소드 전부 묶어져 있음. 
*이클립스 프로젝트> 클린 : 전체 소스파일을 다시 컴파일 할때. (지우는거 아님)
*이클립스 개별파일 컴파일하고 싶을때는 공백주고 저장!

*javac -encoding utf8 -d bin/main -classpath bin/main(같은패키지이기 때문에 여기까지 적어도 됨.) src/main/java/com/eomcs/oop/ex01/Exam0410.java
*javac -encoding utf8 -d bin/main -sourcepath src/main/java src/main/java/com/eomcs/oop/ex01/Exam0410.java(두 개 파일을 컴파일하는 것. class 파일 2개 생김)
*encoding은 소스파일이 utf8로 만들었으니 너도 그렇게 읽어라라는 뜻.
*이클립스 인텔리제이는 컴파일러가 아님. 나 대신 위의 문장을 실행하는 프로그램임.
